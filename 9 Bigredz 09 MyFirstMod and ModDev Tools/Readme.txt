Bigredz MyFirstMod - A sample simple MOD for aspiring MOD authors to use as a starting point.

It also includes a super-useful batch file that will setup a MOD development folder with shortcuts 
to all the different and useful folders in MOO, Steam, and Local as well as creating Env vars so
you can easily go to them on your System.

Include link to Google Docs / Steam Workshop / PDF Mod Gettings Started document
